### a simple app for learning ws_next tools/frameworks:
* graphql
* couchbase
* rewire

uses the couchbase server sample database __travel-sample__

### setup
1. clone repo
1. run `yarn`
1. [setup couchbase server via docker](https://hub.docker.com/r/couchbase/server/)
1. `npm run start`
